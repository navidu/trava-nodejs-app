# Trava (NodeJS App)
- Built using gulp, yarn & nodejs

## How to run
- Git clone this repo
```shell
git clone https://github.com/navidu/trava-nodejs-app.git
```
- install yarn
```shell
npm install yarn -g
```
- install dependencies
```shell
yarn install
```
- install gulp
```shell
npm install gulp
```
- build
```shell
gulp build
```
- run server
```shell
node app.js
```

Go to your browser and type localhost:8080
